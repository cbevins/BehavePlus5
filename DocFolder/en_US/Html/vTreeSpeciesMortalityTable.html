Tree Species Bark Thickness & Mortality Equations

Tree species is used in the BehavePlus MORTALITY and SPOT modules. There are, however, many more species available for MORTALITY than for SPOT. To avoid confusion, there are two tree species variables:

  • Spot tree species, and
  • Mortality tree species.
The same tree species codes are used for both variables. The table below lists all tree species available to the MORTALITY module. The tree species available to the SPOT module are listed in
  • Spot Tree Species

Bark thickness is estimated from the tree species and D.B.H. using equations in FOFEM 5.9: First Order Fire Effects Model (Reinhardt, E.D., R.E. Keane, and J.K. Brown, 1997).

New tree mortality equations are from JFSP Report #05-2-1-105 Delayed Tree Mortality following Fire in Western Conifers (Hood, S., S. Smith, D. Cluck, E. Reinhardt, and K. Ryan, 2008).

Each tree species has a bark thickness equation and a mortality equation associated with it. These are summarized in the tables below.
